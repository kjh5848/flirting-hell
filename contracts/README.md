# API 계약

이 폴더는 Flutter 앱과 Spring Boot 백엔드가 공유할 API 계약을 보관한다.

현재 상태:

- Phase 1에서는 폴더와 위치만 고정한다.
- 실제 OpenAPI 파일은 Spring API v2 구현이 시작되는 Phase 2부터 `contracts/openapi/`에 추가한다.
