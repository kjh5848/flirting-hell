# 결정 0002: 첫 유료 모델은 분석권 패키지로 시작한다

## 상태

확정

## 날짜

2026-04-24

## 배경

플러팅지옥은 사용자가 매일 정해진 루틴으로 쓰기보다, 답장을 보내야 하거나 상대 반응이 헷갈리는 순간에 집중적으로 사용할 가능성이 크다.

초기부터 월 구독을 요구하면 사용자는 아직 반복 가치를 확인하기 전에 결제 부담을 느낄 수 있다.

## 결정

V1의 첫 유료 모델은 `분석권 패키지`로 시작한다.

초기 가격 실험 후보:

- 분석권 30회: 3,900원
- 분석권 50회: 5,900원
- 분석권 100회: 9,900원

## 이유

- 사용자가 급한 순간에 필요한 만큼 구매하기 쉽다.
- 월 구독보다 진입 장벽이 낮다.
- 분석 1회당 비용과 AI 사용 비용을 관리하기 쉽다.
- 반복 구매율을 보면 구독 전환 가능성을 판단할 수 있다.
- 초기 PMF 검증에 더 적합하다.

## 영향

무료 사용자는 하루 3회 분석을 사용할 수 있고, 더 필요할 때 분석권을 구매한다.

월 구독은 초기에 제외하지 않지만, 반복 구매율과 7일 재사용률이 충분히 확인된 뒤 후순위로 실험한다.
