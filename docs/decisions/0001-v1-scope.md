# 결정 0001: V1 범위는 메시지 분석과 답장 추천으로 제한한다

## 상태

확정

## 날짜

2026-04-24

## 배경

초기 아이디어에는 연애 교육, 메시지 분석, 데이트 코칭, 호감 표현, 성인 친밀감 코칭, 스타일 코칭 등 여러 방향이 포함되어 있었다.

처음부터 모든 기능을 만들면 제품의 핵심 가치가 흐려지고, 사용자가 왜 이 앱을 써야 하는지 명확하지 않아진다.

## 결정

플러팅지옥 V1은 `메시지 분석 + 답장 추천 앱`으로 시작한다.

## 이유

- 사용자가 즉시 필요성을 느끼는 문제다.
- 입력과 출력이 명확해서 빠르게 프로토타입을 만들 수 있다.
- AI 기능의 가치를 바로 체감할 수 있다.
- 유료화 가능성이 높다.
- 이후 연애 교육, 데이트 코치, 스타일 코치로 확장하기 쉽다.

## 영향

V1에서는 코스형 연애 교육이나 데이트 전체 설계보다, 대화 맥락 분석과 답장 추천 품질에 집중한다.

향후 확장은 V1 사용 데이터를 보고 결정한다.
