# 결정 0003: MVP는 Next.js 모바일 웹/PWA로 시작한다

## 상태

대체됨

이 결정은 `0004-cloudflare-stack.md`로 대체되었다.

## 날짜

2026-04-24

## 배경

플러팅지옥 V1은 카톡이나 DM 내용을 붙여넣고, AI 분석 결과와 답장 후보를 확인하는 흐름이 핵심이다.

초기부터 네이티브 앱으로 만들면 iOS/Android 배포, 앱스토어 심사, 푸시 알림, 앱 업데이트 등 제품 검증과 직접 관련 없는 비용이 커진다.

## 결정

MVP는 Next.js 기반 모바일 웹/PWA로 시작한다.

추천 스택:

- Next.js
- React
- TypeScript
- Tailwind CSS
- Next.js Route Handlers
- Supabase Postgres
- 분석권 패키지 결제는 별도 결제사 검토 후 연동

## 이유

- 첫 프로토타입을 빠르게 만들 수 있다.
- 모바일 브라우저에서 실제 사용 흐름을 검증할 수 있다.
- 앱스토어 심사 없이 테스트 링크를 공유할 수 있다.
- AI API 키를 서버에서 안전하게 다룰 수 있다.
- 프론트엔드와 백엔드 API를 한 프로젝트에서 관리할 수 있다.

## 영향

V1에서는 네이티브 앱 기능을 목표로 하지 않는다.

사용자 반응, 답장 복사율, 반복 사용률, 분석권 구매 전환율을 확인한 뒤 React Native 또는 Flutter 기반 앱 전환을 검토한다.
