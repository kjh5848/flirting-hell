# 플러팅지옥 앱 디자인 리뉴얼 v2 — 레퍼런스 노트

## 최종 방향

- 선택안: **A. 정돈형 + 따뜻한 포인트**
- 해석: Things 3처럼 정돈된 앱 구조를 기본으로 삼고, Tinder처럼 연애 카테고리에 맞는 rose/coral 포인트만 제한적으로 사용한다.
- 목표: 데이팅앱처럼 과하게 감정적이지 않고, AI 툴처럼 차갑지도 않은 “답장을 고르는 모바일 앱” 인상.

## 관찰한 패턴

- Things 3: 넓은 여백, 낮은 채도, 명확한 카드 계층, 작은 인터랙션 단위가 좋다.
- Tinder: rose/coral 계열 포인트, 단계 진행, 감정적 CTA가 연애 카테고리에 맞다.
- ChatGPT: 대화형 입력 구조는 참고할 수 있지만 AI 제품 인상이 강해 플러팅지옥에는 제한적으로만 참고한다.

## 저작권 원칙

- 외부 앱 스크린샷은 프로젝트에 저장하지 않는다.
- 외부 UI를 픽셀 단위로 복제하지 않는다.
- 레퍼런스에서는 색감, 간격, 정보 계층, 카드 밀도, 화면 흐름만 추상화해서 토큰과 와이어프레임으로 재설계한다.
