# Pillyze Inspired Reference Notes

## Source

- Reference URL: https://wwit.design/2022/12/18/pillyze/
- Usage: 구조, 밀도, 카드 그룹, 리스트 패턴, 하단 CTA, 마이페이지형 설정 화면만 참고한다.

## Copyright Boundary

- 원본 Pillyze 스크린샷, 로고, 카피, 색상, 아이콘, 화면 배치를 repo에 저장하지 않는다.
- 생성 이미지는 플러팅지옥 색상과 제품 문맥으로 재해석한 새 목업이다.
- 구현 시 원본과 1:1로 보이는 화면 복제는 금지한다.

## Observed Patterns To Reinterpret

- 모바일 앱다운 얇은 헤더와 한 화면 안에서 읽히는 정보 밀도.
- 흰색 카드 그룹 안에 설정, 요약, 리스트 항목을 정돈하는 방식.
- 페이지 하단의 큰 CTA가 다음 행동을 분명히 만드는 구조.
- 마이페이지/설정 화면에서 라벨, 현재값, 이동 아이콘을 반복 사용하는 구조.
- 건강/라이프스타일 앱처럼 부담을 낮추는 부드러운 카드와 짧은 안내 문장.

## Flirting Hell Translation

- 건강 정보 대신 `대화 고민`, `상대별 상담방`, `추천 답장`, `조언 수위`를 중심으로 재배치한다.
- 색상은 기존 `DESIGN.md`의 warm off-white, white surface, dark ink, rose accent, coral accent만 사용한다.
- 연애 조언 앱의 안전 경계를 유지하기 위해 압박, 조종, 단정 문구를 화면 구조에서도 강조하지 않는다.
