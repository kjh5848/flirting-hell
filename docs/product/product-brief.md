# 플러팅지옥 제품 브리프

## 제품 한 줄 설명

플러팅지옥은 연애 초보가 상대와 나눈 메시지를 입력하면, AI가 분위기를 분석하고 지금 보내기 좋은 답장을 추천해주는 연애 대화 코치다.

## 문제

연애를 처음 시작하는 사람은 상대의 답장이 호감인지 예의인지 구분하기 어렵다. 답장을 보내기 전에도 너무 가벼워 보일까, 부담스럽게 느껴질까, 대화가 끊길까 고민한다.

이 문제는 단순한 문장 생성 문제가 아니라, 상황 판단과 감정 조절의 문제다.

## 목표 사용자

첫 버전의 목표 사용자는 `연애 초보 전체`다.

성별을 기준으로 제품을 나누기보다, 온보딩에서 다음 정보를 받아 개인화한다.

- 사용자의 연애 경험 수준
- 사용자가 호감을 느끼는 상대
- 현재 관계 단계
- 대화 목적
- 원하는 플러팅 강도
- 사용자의 평소 말투와 톤앤매너

## V1 핵심 기능

1. 사용자가 카톡, DM, 문자 내용을 붙여넣는다.
2. AI가 상대의 반응과 대화 분위기를 분석한다.
3. 현재 상태를 `호감 있음`, `애매함`, `부담 가능성`, `잠시 쉬어야 함`으로 보여준다.
4. 지금 보낼 답장 후보 3개를 추천한다.
5. 각 답장에 `순한맛`, `설렘맛`, `직진맛` 강도를 붙인다.
6. 답장 후보를 사용자의 말투와 톤앤매너에 맞게 다듬는다.
7. 보내면 안 좋은 말과 그 이유를 알려준다.
8. 다음 행동을 `대화 이어가기`, `약속 잡기`, `한 템포 쉬기` 중에서 추천한다.

## V1 비목표

- 상대를 조종하거나 압박하는 방법 제공
- 성적 만남을 성사시키는 전술 제공
- 특정 성별을 일반화하는 공략법 제공
- 동의 없는 관계 진전 유도
- 사적인 메시지를 장기 보관하는 기능

## 제품 원칙

- 많이 보내는 것보다 적절하게 보내는 것을 돕는다.
- 상대의 거절 신호를 무시하지 않는다.
- 사용자의 자신감을 키우되 과한 확신을 주지 않는다.
- 답장 추천은 항상 여러 강도로 제공한다.
- 답장은 사용자의 평소 말투와 너무 다르게 보이지 않아야 한다.
- 민감한 관계 상황에서는 쉬어가는 선택지를 제공한다.

## 성공 지표 후보

- 분석 후 답장 후보를 복사한 비율
- 첫 분석 후 7일 내 재사용률
- 사용자가 `도움 됨`으로 평가한 분석 비율
- `부담 가능성` 경고가 표시된 뒤 사용자가 강도를 낮춘 비율
- 유료 전환 전까지 평균 분석 횟수
