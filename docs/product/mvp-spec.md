# 플러팅지옥 MVP 설계서

## 목적

V1의 목적은 사용자가 상대와 나눈 메시지를 붙여넣고, 지금 대화의 분위기와 다음 답장을 빠르게 판단하도록 돕는 것이다.

제품은 `상대에게 더 많이 플러팅하는 도구`가 아니라 `부담을 줄이고 호감을 자연스럽게 표현하는 대화 코치`로 동작한다.

또한 제품은 사용자의 이상형과 연애 스타일을 기준으로 상대를 분석하되, 특정 상대를 만나도 되는지 금지하거나 단정하지 않는다. 사용자가 이상형과 다른 사람에게 끌릴 수 있고, 외모나 분위기 때문에 계속 만나고 싶을 수도 있다는 현실을 인정한다.

## 핵심 사용자 흐름

1. 사용자가 온보딩에서 연애 경험, 관계 단계, 대화 목적을 선택한다.
2. 사용자가 선호하는 이상형과 원하는 연애 스타일을 선택한다.
3. 사용자가 카톡, DM, 문자 내용을 붙여넣는다.
4. AI가 사용자의 메시지에서 말투와 톤앤매너를 분석한다.
5. 사용자가 원하는 답장 강도와 조언 수위를 선택한다.
6. AI가 대화 분위기, 상대 반응, 성향 적합도, 위험 신호를 분석한다.
7. AI가 사용자의 말투를 반영한 답장 후보 3개와 피해야 할 말을 제안한다.
8. 사용자가 답장을 복사하거나 다시 생성한다.
9. 사용자가 결과에 `도움 됨` 또는 `별로임`을 남긴다.

## 첫 화면

첫 화면은 마케팅용 랜딩 페이지가 아니라 바로 분석을 시작하는 앱 화면으로 구성한다.

필수 요소:

- 앱 이름: `플러팅지옥`
- 짧은 카피: `플러팅 무한루프 탈출. 지금 보낼 한마디를 찾다.`
- 메시지 입력 영역
- 관계 단계 선택
- 답장 강도 선택
- 조언 수위 선택
- 내 말투 분석 또는 설정
- 분석 시작 버튼

## 온보딩 입력값

온보딩은 길지 않아야 한다. 첫 사용자는 바로 메시지를 분석하고 싶어 하므로, 선택지는 최소화한다.

필수 입력:

- 연애 경험: `처음`, `익숙하지 않음`, `보통`
- 관계 단계: `처음 연락`, `썸`, `데이트 전`, `데이트 후`, `관계 회복`
- 대화 목적: `대화 이어가기`, `호감 표현`, `약속 잡기`, `실수 만회`, `상대 마음 확인`
- 답장 강도: `순한맛`, `설렘맛`, `직진맛`
- 조언 수위: `응원 위주`, `균형 조언`, `현실 체크`
- 말투 반영: `자동 분석`, `직접 설정`, `이번엔 반영 안 함`

선택 입력:

- 사용자의 말투: `다정한`, `장난스러운`, `담백한`, `적극적인`
- 피하고 싶은 톤: `느끼함`, `부담스러움`, `너무 진지함`, `가벼움`
- 선호하는 상대 스타일: `상냥한`, `애교 있는`, `표현이 많은`, `차분한`, `시크한`, `털털한`, `대화가 잘 통하는`
- 어려워하는 상대 스타일: `무뚝뚝한`, `연락이 느린`, `표현이 적은`, `너무 가벼운`, `감정 기복이 큰`

## 이상형과 연애 스타일 분석

사용자의 이상형은 고정된 정답이 아니다. 사용자는 시간이 지나며 원하는 연애 스타일이 바뀔 수 있고, 이상형과 다르더라도 계속 만나고 싶은 사람을 선택할 수 있다.

AI는 상대를 `만나라`, `만나지 마라`로 판정하지 않는다. 대신 사용자가 원하는 연애 스타일과 현재 상대의 대화 방식 사이에 어떤 차이가 있는지 설명한다.

예시:

`상대는 시크하고 담백한 스타일로 보입니다. 사용자는 상냥하고 표현이 많은 연애를 선호하므로, 관계가 깊어질수록 외로움을 느낄 수 있어요. 다만 이 차이가 반드시 나쁘다는 뜻은 아닙니다. 계속 보고 싶은 마음이 있다면, 상대가 애정 표현을 어떻게 하는 사람인지 천천히 확인해보세요.`

### 적합도 상태값

- `잘 맞을 가능성 있음`
- `매력은 있지만 피로할 수 있음`
- `내 이상형과 차이가 큼`
- `더 확인 필요`

### 조언 수위

사용자는 조언 수위를 선택할 수 있다.

- `응원 위주`: 경고는 짧게 제공하고, 사용자의 선택을 존중하는 답장을 우선 추천한다.
- `균형 조언`: 매력 포인트와 현실적 리스크를 함께 설명한다.
- `현실 체크`: 사용자가 놓칠 수 있는 위험 신호와 장기적 피로 가능성을 더 분명하게 알려준다.

모든 수위에서 마지막 톤은 사용자의 선택을 존중해야 한다.

예시:

`선택은 사용자의 몫이에요. 이 사람이 계속 보고 싶은 사람이라면, 무리하게 단정하지 말고 작은 대화로 천천히 확인해보세요. 플러팅지옥은 당신의 연애를 응원합니다.`

## 말투와 톤앤매너 분석

플러팅지옥의 답장은 사용자의 평소 말투를 반영해야 한다. 좋은 문장이라도 사용자가 평소 쓰지 않는 표현이면 상대에게 어색하게 보일 수 있다.

AI는 사용자가 붙여넣은 대화에서 `나:`에 해당하는 메시지를 기준으로 말투를 분석한다.

분석 항목:

- 존댓말과 반말 비율
- 문장 길이
- 이모티콘과 이모지 사용 빈도
- 웃음 표현: `ㅋㅋ`, `ㅎㅎ`, `하하`
- 질문 빈도
- 장난기
- 다정함
- 직진도
- 진지함

답장 후보는 모두 이 말투 프로필을 반영한다.

예시:

`사용자는 짧고 장난스러운 말투를 쓰는 편이므로, 답장도 길게 감정을 설명하기보다 가볍게 호감을 남기는 방식이 자연스럽습니다.`

사용자는 말투 반영 방식을 선택할 수 있다.

- `자동 분석`: 현재 대화에서 사용자의 말투를 추정한다.
- `직접 설정`: 사용자가 원하는 말투를 직접 고른다.
- `이번엔 반영 안 함`: 일반적인 추천 답장만 받는다.

원문 대화는 장기 보관하지 않는 것을 기본으로 하고, 저장이 필요하면 요약된 말투 프로필만 저장한다.

## 메시지 분석 입력

사용자는 대화 전문을 붙여넣는다.

입력 가이드:

- 이름, 전화번호, 주소 같은 개인정보는 지우도록 안내한다.
- 메시지는 최근 대화 10~30줄 정도를 권장한다.
- 상대와 사용자의 말이 구분되지 않으면 AI가 먼저 구분을 요청한다.

## AI 출력 포맷

AI 응답은 매번 같은 구조로 제공한다.

### 1. 현재 분위기

가능한 상태값:

- `호감 있음`
- `애매함`
- `부담 가능성`
- `잠시 쉬어야 함`

출력 내용:

- 한 줄 요약
- 판단 근거 2~3개
- 확신도: `낮음`, `보통`, `높음`

### 2. 내 스타일과의 적합도

사용자가 설정한 이상형과 연애 스타일을 기준으로 상대의 대화 방식을 비교한다.

가능한 상태값:

- `잘 맞을 가능성 있음`
- `매력은 있지만 피로할 수 있음`
- `내 이상형과 차이가 큼`
- `더 확인 필요`

출력 내용:

- 한 줄 요약
- 잘 맞는 부분
- 어긋날 수 있는 부분
- 계속 만나고 싶을 때 확인할 질문
- 조언 수위에 맞춘 안내 문장

### 3. 내 말투 분석

사용자의 메시지를 기준으로 답장에 반영할 톤앤매너를 요약한다.

출력 내용:

- 말투 요약
- 평소 문장 길이
- 웃음 표현과 이모티콘 사용
- 플러팅할 때 유지해야 할 자연스러운 표현
- 피해야 할 어색한 표현

### 4. 지금 보내기 좋은 답장

답장 후보는 3개를 제공한다.

- `순한맛`: 안전하고 부담 없는 답장
- `설렘맛`: 호감을 은근히 드러내는 답장
- `직진맛`: 상대 반응이 좋을 때만 쓸 수 있는 답장

각 답장에는 다음 정보를 붙인다.

- 추천 문장
- 이 답장이 좋은 이유
- 사용하면 좋은 상황
- 부담도: `낮음`, `중간`, `높음`
- 말투 반영 설명

### 5. 보내면 위험한 말

사용자가 보내지 않는 것이 좋은 표현을 알려준다.

포함 내용:

- 피해야 할 문장 예시
- 위험한 이유
- 더 나은 대체 표현

### 6. 다음 행동

가능한 추천 행동:

- `대화 이어가기`
- `약속 잡기`
- `한 템포 쉬기`
- `사과하기`
- `관심사 질문하기`

각 행동에는 짧은 이유를 붙인다.

## 안전 정책

플러팅지옥은 다음을 제공하지 않는다.

- 상대를 압박하거나 조종하는 방법
- 거절 의사를 무시하는 답장
- 집착, 감시, 추적을 유도하는 조언
- 성적 만남을 강하게 밀어붙이는 표현
- 미성년자 대상 성적 조언
- 개인정보 수집, 저장, 악용을 유도하는 조언

민감한 상황에서는 답장보다 `한 템포 쉬기`, `사과하기`, `명확한 동의 확인`을 우선 추천한다.

이상형 적합도 분석은 사용자의 선택을 대신하지 않는다. 경고가 필요한 경우에도 `절대 만나지 마세요`처럼 단정하지 않고, 사용자가 확인해야 할 차이와 질문을 제안한다.

## 예외 상황

### 메시지가 너무 짧을 때

AI는 분석을 단정하지 않고 추가 맥락을 요청한다.

예시:

`대화가 짧아서 호감 여부를 단정하기 어려워요. 상대가 마지막에 어떤 말투였는지, 이전에 약속을 잡은 적이 있는지 알려주세요.`

### 상대가 불편해 보일 때

AI는 플러팅 답장보다 부담을 낮추는 답장을 추천한다.

예시:

`지금은 더 밀기보다 가볍게 마무리하는 쪽이 안전해요.`

### 사용자가 과하게 직진하려 할 때

AI는 강도를 낮춘 대체 문장을 제공한다.

예시:

`직진맛은 지금 대화 분위기에서는 부담이 될 수 있어요. 설렘맛으로 낮춰볼게요.`

## 무료와 유료 구분 초안

무료:

- 하루 3회 메시지 분석
- 답장 후보 3개
- 기본 분위기 판단
- 기본 이상형/연애 스타일 적합도
- 현재 대화 기준 말투 자동 분석
- 보내면 위험한 말 1개

유료:

- 분석권 기반 추가 분석
- 말투 개인화
- 말투 프로필 여러 개 저장
- AI 티 나는 문장 자동 제거
- 이상형 프로필 여러 개 저장
- 조언 수위 개인화
- 대화 히스토리 기반 코칭
- 데이트 약속 잡기 전용 모드
- 실수 만회 모드
- 고급 분석: 상대의 관심사, 대화 패턴, 다음 타이밍

V1의 첫 유료 모델은 월 구독이 아니라 `분석권 패키지`로 시작한다. 결제 제공자는 Polar를 사용하고, 결제 완료 webhook을 Workers가 받아 D1의 분석권 잔액을 증가시킨다. 자세한 과금 기준과 측정 지표는 `docs/product/monetization-metrics.md`에서 관리한다.

- 답장 복사율
- 답장 재생성률
- `도움 됨` 평가 비율
- 첫 사용 후 7일 내 재방문율
- 무료 사용자의 유료 전환율
