# 플러팅지옥 말투 프로필 설계

## 목적

플러팅지옥의 답장은 좋은 문장보다 `사용자답게 들리는 문장`이어야 한다.

사용자가 카톡이나 DM을 붙여넣으면, AI는 대화 분위기뿐 아니라 사용자의 평소 말투와 톤앤매너를 분석해 답장 후보에 반영한다.

## 핵심 원칙

- 답장은 사용자의 말투에서 벗어나지 않아야 한다.
- AI가 만든 티가 나는 과한 표현은 줄인다.
- 사용자의 원래 매력을 살리되, 부담스럽거나 오해될 표현은 다듬는다.
- 말투 분석은 상대를 속이기 위한 위장이 아니라, 사용자가 자연스럽게 표현하도록 돕는 보정이다.
- 원문 메시지를 장기 보관하기보다, 요약된 말투 프로필만 저장하는 방향을 기본으로 한다.

## 말투 분석 입력

말투 분석은 두 방식으로 제공한다.

### 1. 자동 분석

사용자가 붙여넣은 대화에서 `나:`로 표시된 메시지를 기준으로 말투를 분석한다.

분석 항목:

- 존댓말과 반말 비율
- 문장 길이
- 이모티콘과 이모지 사용 빈도
- 웃음 표현: `ㅋㅋ`, `ㅎㅎ`, `하하`
- 감탄사 사용: `아`, `오`, `헐`, `진짜`
- 질문 빈도
- 장난기
- 다정함
- 직진도
- 진지함
- 맞춤법과 띄어쓰기의 자연스러운 수준

### 2. 수동 설정

사용자가 직접 원하는 답장 톤을 고를 수 있다.

선택값:

- 기본 말투: `반말`, `존댓말`, `섞어서`
- 문장 길이: `짧게`, `보통`, `길게`
- 분위기: `담백한`, `다정한`, `장난스러운`, `설레는`, `차분한`
- 표현 강도: `은근하게`, `보통`, `확실하게`
- 이모티콘 사용: `안 씀`, `가끔`, `자주`
- 웃음 표현: `안 씀`, `ㅋㅋ`, `ㅎㅎ`, `섞어서`

## 말투 프로필

AI는 사용자의 말투를 다음처럼 요약한다.

```json
{
  "speechLevel": "반말",
  "sentenceLength": "짧게",
  "warmth": "중간",
  "playfulness": "높음",
  "directness": "낮음",
  "emojiUsage": "가끔",
  "laughStyle": "ㅋㅋ",
  "signaturePatterns": [
    "문장을 짧게 끊어 말함",
    "가벼운 장난을 섞음",
    "호감 표현은 직접적으로 많이 하지 않음"
  ],
  "avoidPatterns": [
    "갑자기 너무 진지한 고백 문장",
    "평소보다 긴 감정 설명",
    "사용자가 잘 쓰지 않는 과한 애교"
  ]
}
```

## 답장 생성 방식

답장 후보 3개는 모두 사용자의 말투 프로필을 반영한다.

각 답장은 다음 기준을 만족해야 한다.

- 사용자가 평소 쓰지 않을 법한 표현을 피한다.
- 원래 말투보다 플러팅 강도가 올라가도 어색하지 않아야 한다.
- 상대에게 보내기 전 사용자가 직접 고칠 수 있는 여지를 남긴다.
- 말투를 너무 따라 하느라 의미가 약해지면 안 된다.

## 답장 카드 표시

각 답장 카드에는 말투 반영 정보를 짧게 보여준다.

예시:

- `평소처럼 짧고 장난스럽게`
- `다정하지만 과하지 않게`
- `직진맛이지만 말투는 담백하게`

## 말투 차이 경고

AI가 추천한 문장이 사용자의 평소 말투와 크게 다르면 경고한다.

예시:

`이 문장은 의미는 좋지만 평소 말투보다 너무 진지해 보여요. 조금 더 짧고 가볍게 바꿔볼게요.`

## 개인정보 원칙

- 원문 대화는 분석에만 사용한다.
- 기본 정책은 원문을 장기 보관하지 않는 것이다.
- 저장이 필요하면 사용자의 명시적 동의를 받는다.
- 저장 대상은 원문이 아니라 요약된 말투 프로필을 우선으로 한다.

## 유료 기능 확장

- 말투 프로필 여러 개 저장
- 상대별 말투 조정
- 카톡식, 인스타 DM식, 소개팅 앱식 답장 모드
- 내 말투 유지하면서 플러팅 강도만 올리기
- AI 티 나는 문장 자동 제거
